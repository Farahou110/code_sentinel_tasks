# code_sentinel_tasks

A collection of Python coding challenges and solutions.

## Getting Started

### Prerequisites

- Python 3.8 or higher  
- [pip](https://pip.pypa.io/en/stable/installation/) (Python package manager)

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/your-username/code_sentinel_tasks.git
   cd code_sentinel_tasks/python
   ```

2. **(Optional) Create a virtual environment:**
   ```sh
   python -m venv venv
   venv\Scripts\activate  # On Windows
   # source venv/bin/activate  # On macOS/Linux
   ```

3. **Install dependencies:**
   ```sh
   pip install -r requirements.txt
   ```

## Running the Code

- To run a specific Python script:
  ```sh
  python script_name.py
  ```

- To run all tests (if you have tests set up):
  ```sh
  python -m unittest discover
  ```
